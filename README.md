<a name="installation"></a>

## 2. Installation

The installation process is fast and streamlined, allowing you to focus on what matters most: creating.

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/TemporalLabsLLC-SOL/TemporalPromptGenerator.git
   cd TemporalPromptGenerator
   ```

2. **Run Setup**:
   ```bash
   python setup.py
   ```

3. **Follow On-Screen Prompts**:
   Upon your first run, a popup will appear asking for your Hugging Face token. Enter it, and the Temporal Prompt Engine will be ready to go.

<a name="setting-up-the-environment"></a>

## 3. Setting Up the Environment

The Temporal Prompt Engine ensures an easy environment setup. When you run `python setup.py`, the script automatically checks for the following:

- **Hugging Face Token**: If not already set, a popup will request your Hugging Face token for seamless API integration.
- **Output Directory**: Where generated media will be stored.
- **Prompts Directory**: Optionally, link your ComfyUI prompts folder for integrated prompt generation.

<a name="how-to-use-the-temporal-prompt-engine"></a>

## 5. How to Use the Temporal Prompt Engine

### **Step-by-Step Guide**

1. **Run the Application**:
   ```bash
   python TemporalPromptEngine.py
   ```

2. **Enter Your Scene Concept**:
   Input your creative scene idea (up to 450 characters).
   - **Example**: "A moonlit forest with ethereal mist swirling around ancient stone ruins."

3. **Configure Video and Audio Options**:
   Customize elements like:
   - **Video**: Framing, camera movement, lighting.
   - **Audio**: Intensity, layering, focus on ambient sounds.

4. **Generate Prompts**:
   The engine will tailor the prompts to match your settings, producing cohesive video and audio outputs.

5. **Integrate with Ollama**:
   The engine automatically works with Ollama models for video creation based on your prompts.

6. **Save and Export**:
   All media files will be saved in the output directory, ready for further refinement or distribution.

## 6. Settings Guide

### **Video Prompt Settings**

1. **Theme**: Sets the overall mood and genre.
2. **Art Style**: Defines the visual aesthetics.
3. **Lighting**: Influences the scene's mood.
4. **Camera Movement**: Adds dynamism to your scenes.
5. **Resolution**: Choose between 4K, UHD, or other formats.
6. **Framing**: Select close-ups, wide shots, and more.

### **Audio Prompt Settings**

1. **Layer Intensity**: Adjust the complexity of sound layers.
2. **Sound Focus**: Exclude music or focus on ambient sounds.
3. **Holiday Mode**: Add seasonal or festive elements.
4. **Remix Mode**: Introduce unpredictability for experimental results.

<a name="troubleshooting"></a>

## 10. Troubleshooting

### **Common Issues and Solutions**

1. **Hugging Face Token Error**:
   - If your token is incorrect, re-run the application and input the correct token.
  
2. **Video or Audio Generation Issues**:
   - Ensure your machine meets the minimum system requirements (16 GB RAM recommended).
   - For any errors, run the included `troubleshoot.py` script.

3. **File Permissions**:
   - Ensure read/write permissions are set for your directories.
